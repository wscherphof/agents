# Extract Merged Instructions Into An Imported File — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Move the project repo's mirrored `CLAUDE.md` content out of the root `CLAUDE.md`'s inline managed block and into its own `@import`ed file, `.claude/merged-agent-instructions.md`, simplifying `merge_claude_md`.

**Architecture:** `merge_claude_md` writes the assembled per-layer body to `.claude/merged-agent-instructions.md` (with a leading "do not edit" marker comment) and no longer reads or writes the root `CLAUDE.md`. The root `CLAUDE.md` becomes fully hand-maintained, gaining a static `@.claude/merged-agent-instructions.md` import line in place of the old `<!-- BEGIN/END MERGED AGENT INSTRUCTIONS -->` block. A committed marker-comment-only placeholder ships on `main` so the import never dangles.

**Tech Stack:** Bash + jq/Python (existing merge script), Markdown (CLAUDE.md memory imports), Git.

## Global Constraints

- Generated file path is exactly `.claude/merged-agent-instructions.md` (relative to the agents repo root).
- The generated file's first line is exactly: `<!-- auto-generated by merge-agent-settings.sh; do not edit -->`.
- The merge script must **never delete** the generated file — on an empty body it writes the marker-comment-only placeholder, so the root `CLAUDE.md` `@import` never dangles.
- The import token in the root `CLAUDE.md` must be a bare `@.claude/merged-agent-instructions.md` on its own line.
- `merge_claude_md` must not read or write `$DEST/CLAUDE.md` at all.
- Per-layer body format is unchanged: `## From <account>/<repo> (<label>)` header, blank line, the layer's `CLAUDE.md`, trailing newline.
- Do NOT add legacy-block cleanup to the script (migration is out of scope per the spec).

```bash
# --- 7. project CLAUDE.md -> separate imported file -------------------------
# Write the project repo's mirrored CLAUDE.md content (root layer, optionally
# + component layer) to its own file, .claude/merged-agent-instructions.md,
# which the agents repo's hand-maintained root CLAUDE.md @imports. The root
# CLAUDE.md is never read or written here. When no source layer has a
# CLAUDE.md, a marker-comment-only placeholder is written (never deleted) so
# the root CLAUDE.md's @import never dangles.
MERGED_MARKER='<!-- auto-generated by merge-agent-settings.sh; do not edit -->'

merge_claude_md() {
  local out="$DEST/.claude/merged-agent-instructions.md" body i layer label f
  body="$(mktemp)"
  TMPFILES+=("$body")
  : >"$body"
  for i in "${!SRC_LAYERS[@]}"; do
    layer="${SRC_LAYERS[$i]}"
    label="${SRC_LABELS[$i]}"
    f="$layer/CLAUDE.md"
    [ -s "$f" ] || continue
    {
      printf '## From %s/%s (%s)\n\n' "$AGENTS_GIT_ACCOUNT" "$AGENTS_GIT_REPO" "$label"
      cat "$f"
      printf '\n'
    } >>"$body"
  done

  mkdir -p "$(dirname "$out")"
  {
    printf '%s\n' "$MERGED_MARKER"
    if [ -s "$body" ]; then
      printf '\n'
      cat "$body"
    fi
  } >"$out"
}
```

```bash
#!/usr/bin/env bash
set -euo pipefail

MERGED_MARKER='<!-- auto-generated by merge-agent-settings.sh; do not edit -->'
AGENTS_GIT_ACCOUNT=acct
AGENTS_GIT_REPO=repo

work="$(mktemp -d)"
DEST="$work/dest"; mkdir -p "$DEST/.claude"

# repro of the function body (kept in sync with merge_claude_md)
run_merge() {
  local out="$DEST/.claude/merged-agent-instructions.md" body i layer label f
  body="$(mktemp)"
  : >"$body"
  for i in "${!SRC_LAYERS[@]}"; do
    layer="${SRC_LAYERS[$i]}"; label="${SRC_LABELS[$i]}"; f="$layer/CLAUDE.md"
    [ -s "$f" ] || continue
    { printf '## From %s/%s (%s)\n\n' "$AGENTS_GIT_ACCOUNT" "$AGENTS_GIT_REPO" "$label"
      cat "$f"; printf '\n'; } >>"$body"
  done
  mkdir -p "$(dirname "$out")"
  { printf '%s\n' "$MERGED_MARKER"
    if [ -s "$body" ]; then printf '\n'; cat "$body"; fi; } >"$out"
  rm -f "$body"
}

# case A: a root layer with a CLAUDE.md
root="$work/root"; mkdir -p "$root"
printf 'Root project rules.\n' >"$root/CLAUDE.md"
SRC_LAYERS=("$root"); SRC_LABELS=("root")
run_merge
out="$DEST/.claude/merged-agent-instructions.md"
head -1 "$out" | grep -qx "$MERGED_MARKER"            && echo "PASS: marker first line"     || { echo FAIL marker; exit 1; }
grep -q '## From acct/repo (root)' "$out"             && echo "PASS: layer header present"  || { echo FAIL header; exit 1; }
grep -q 'Root project rules.' "$out"                  && echo "PASS: layer body present"    || { echo FAIL body; exit 1; }

# case B: no layer has a CLAUDE.md -> placeholder only, file still exists
empty="$work/empty"; mkdir -p "$empty"
SRC_LAYERS=("$empty"); SRC_LABELS=("root")
run_merge
[ -f "$out" ]                                         && echo "PASS: file not deleted"      || { echo FAIL deleted; exit 1; }
[ "$(grep -c '.' "$out")" -eq 1 ]                     && echo "PASS: placeholder only"      || { echo FAIL placeholder; exit 1; }

rm -rf "$work"
echo "ALL PASS"
```
